This folder contains the ASP.NET Core MVC views.

For an overview, see:

    https://docs.microsoft.com/en-us/aspnet/core/mvc/views/overview

Folders (except for "Shared") must be named after their associated controller (e.g. "Home" for the "HomeController").
Views should be named after their associated action (e.g. "Privacy.cshtml" for "HomeController.Privacy()"). For more
details on automatic view finding, see:

    https://docs.microsoft.com/en-us/aspnet/core/mvc/views/overview#view-discovery

All views are "Razor" files (.cshtml). For details on the syntax, see:

    https://docs.microsoft.com/en-us/aspnet/core/mvc/views/razor
